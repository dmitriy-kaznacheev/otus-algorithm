# Базовые структуры данных

## Цель
Реализовать различные структуры данных;
сравнить эффективность различных подходов;
выбрать подходящую структуру данных под конкретную задачу.

## Описание
Создать разне алгоритмы для реализации динамического массива и сравненить их производительность. 
Создать приоритетную очередь или разреженный массив.
